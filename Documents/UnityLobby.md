# Unity Lobby
## Overview
Unity provides a suite of multiplayer tools for multiplayer games, including a Lobby, Matchmaking, and Relay API through the Multiplayer Services SDK.

The CodeMonkey tutorial for multiplayer utilizes these tools/services under the free tier to enable multiplayer functionality in the game.
