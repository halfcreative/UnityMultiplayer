# Unity Multiplayer Tutorial

This project is the result of following the "Kitchen Chaos Multiplayer Course" tutorial by Code Monkey. The purpose of this tutorial is to learn how to create multiplayer games in Unity using the official Netcode for GameObjects package. This project serves as a hands-on experience to understand the fundamental concepts of networking in game development.

## More Information

This project is built with Unity and C#. It explores the creation of a multiplayer game from a single-player version. The key concepts and technologies covered in this tutorial include:

-   **Unity's Netcode for Game Objects:** The core of the multiplayer functionality.
-   **Server Authoritative vs. Client Authoritative Models:** Understanding the difference and choosing the right model for your game.
-   **Remote Procedure Calls (RPCs):** To communicate between the server and clients.
-   **NetworkVariables:** To synchronize data between clients.
-   **Player Character Synchronization:** Including movement and animations.
-   **Game State Synchronization:** Managing game states like countdown, game start, and game over across the network.
-   **Lobby and Relay Services:** Using Unity's services to create lobbies and connect players.
-   **Character Selection:** Implementing a character selection screen for multiplayer.
